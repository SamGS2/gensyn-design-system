How to use this design system

For new Gensyn-branded apps — manual setup or with Claude Code / Cursor.

Manual setup

  1. Clone or copy this repo: github.com/SamGS2/gensyn-design-system
  2. Copy css/ and assets/ into your project.
  3. Import in your main stylesheet: 
    @import './styles/gensyn-tokens.css';
@import './styles/gensyn-components.css';
  4. Add class="gensyn-page" on <body> for the lavender page background.
  5. Wrap content in .gensyn-container and white panels in .gensyn-card.
  6. Optional animated intake background: iframe URL in the Background section below.

Claude Code / Cursor

  1. Add the kit inside your repo (submodule or copied packages/gensyn-design-system/ folder).
  2. Copy CLAUDE.md.template from this repo to your project root as CLAUDE.md (or a Cursor rule).
  3. Point Claude at this style guide: https://samgs2.github.io/gensyn-design-system/
  4. Example prompt: 
    Read CLAUDE.md and packages/gensyn-design-system/README.md.
Build [your screen] using Gensyn classes only — gensyn-page,
gensyn-card, gensyn-btn-primary. Use CSS variables from
gensyn-tokens.css. Do not invent new colours.
  5. Do not paste the whole FitFinder app into chat — only this design-system folder + your feature request.

Page background

Every screen uses a fixed lavender gradient on html and body. White .gensyn-card panels sit on top for forms and results.

Gradient links (copy for new projects)

Animated (optional, intake-style): https://colorflow-embed.b-cdn.net/embed.html#e=wvk6jmkA

Static (default, all pages): linear-gradient(160deg, #edf0fb 0%, #f5eef9 100%) — token --gensyn-bg-gradient

Page gradient fill

#edf0fb → #f5eef9 · 160deg · --gensyn-bg-gradient

This is the full-page background users see around white content cards — not a separate white panel.
